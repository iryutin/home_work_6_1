# Домашняя работа по реализации бэкенда сайта

## Описание:
### Созданы приложения:
- **catalog** Базовое приложение
- - Реализованы два контролера для отображения двух страниц home и contacts
- - В контролере contacts реализована обработка POST запроса




## Установка:

1. Клонируйте репозиторий:
```
https://github.com/iryutin/home_work_6_1
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
